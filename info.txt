WM PACKAGES:

wlsunset - night light, redshift alternative. can also use gammastep -- ! ONLY IF NOT USING HYPRLAND, hyprsunset
brightnessctl - used for keyboard audio/brightness control -- ! ONLY IF NOT USING HYPRLAND, hyprsunset
grim, slurp - screenshot util
wl-clipboard - wayland clipboard cli utils like wl-copy wl-paste
cliphist - used with wl-clipboard and wofi to get clipboard history
policykit-1-gnome - policy kit for auth in gnome -- ! HYPRLAND HAS ITS OWN
nwg-look - set gtk theme
playerctl - used in WMs for play/stop/skip videos
mako - notifications deamon ( on ubuntu its mako-notifier )
libnotify - notify-send command for notification scripts
xdg-desktop-portal-hyprland - An XDG Desktop Portal that lets other applications communicate with the hyprland


LY DISPLAY MANAGER:

for ly to set XDG values for wms ex. XDG_CURRENT_DESKTOP, make .desktop files in /usr/share/wayland-sessions


WM COMMANDS:

use nmtui for network connection in tty
use bluetoothctl for bluetooth connection in tty
